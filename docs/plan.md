# 竞品情报自动抓取方案

## 目标
从AI行业新闻源自动抓取竞品动态，AI分类后写入飞书多维表格，前端实时展示。

## 数据源
1. **36kr AI板块** - https://36kr.com/search/articles/AI
2. **晚点LatePost** - https://www.latepost.com/
3. **Tech星球** - https://www.techplanet.cc/
4. **量子位** - https://www.qbitai.com/
5. **机器之心** - https://www.jiqizhixin.com/
6. **虎嗅AI** - https://www.huxiu.com/

## 架构

### 1. 飞书多维表格（竞品情报库）
- 公司（文本）
- 类型（单选：组织架构/人才流动/融资/产品/招聘/合作）
- 标题（文本）
- 摘要（多行文本）
- 日期（日期）
- 来源链接（超链接）
- 影响等级（单选：高/中/低）
- 关键人物（文本）
- 备注（多行文本）
- 状态（单选：待审核/已发布/已忽略）

### 2. 爬虫脚本（Python）
- 定期爬取各新闻源
- 用LLM/关键词匹配提取：公司名、事件类型、影响等级
- 去重（基于标题+日期）
- 写入飞书多维表格

### 3. Cron 任务
- 每4小时执行一次（一天6次）
- 早8点、12点、16点、20点、0点、4点

### 4. 前端更新
- API路由从飞书拉取数据
- 实时展示，无需mock

## 实现步骤
1. 创建飞书多维表格
2. 写爬虫脚本 + 分类逻辑
3. 部署cron
4. 更新前端API从飞书读取
5. 测试+上线
